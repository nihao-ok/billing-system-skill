# 对账单生成系统 Skill

WorkBuddy 零配置通用版。首次使用自动注册试用账户。

## 安装

1. 下载 [SKILL.md](https://raw.githubusercontent.com/nihao-ok/billing-system-skill/main/SKILL.md)
2. 放到 
3. 重启 WorkBuddy
4. 对 WorkBuddy 说「生成对账单」

## 首次使用

WorkBuddy 会自动注册试用账户（3 次免费配额）。无需手动配置。

## 计费

- 试用：3 次免费
- 正式账户：按配额计费
- 余额不足联系管理员（Claire）

## 支持

联系管理员开通正式账户。
